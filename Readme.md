## 📌 O que é este projeto?
Este é o site institucional da **Menephly Freelancer**, uma empresa focada em **programação mobile, desenvolvimento de sites web e soluções com inteligência artificial**.  
O objetivo é apresentar serviços e cases de forma moderna, interativa e profissional, transmitindo inovação e tecnologia.

## 🎯 Principais Recursos
- **Design moderno e responsivo**  
- **Animações interativas** (partículas, ripple effect, energia dinâmica)  
- **Seções claras e objetivas**: Sobre, Serviços, Cases e Contato  
- **Integração direta com WhatsApp** para atendimento rápido  
- **Identidade visual personalizada** com logo e cores exclusivas  

## ⚙️ Tecnologias Utilizadas na criação deste site
- HTML5  
- CSS3 (gradientes, animações, responsividade)  
- JavaScript (ES6)  
- Node.js + NPM (gerenciamento de dependências)
